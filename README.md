# 🎓 AI Learning Buddy

An AI-powered learning assistant built using **Python**, **Google Gemini API**, and **Streamlit**. This application helps users learn any topic in different ways by generating AI-powered explanations, answering questions, providing real-life examples, or creating quizzes.

## 📌 Overview

AI Learning Buddy is designed to make learning more interactive and personalized. Instead of receiving the same type of response every time, users can choose how they want to learn a topic.

Simply enter a topic, select a learning mode, and let Gemini AI generate content tailored to your choice.

## ✨ Features

* 📖 **Explain** – Receive a clear and easy-to-understand explanation of any topic.
* 💬 **Ask Anything** – Ask questions related to the topic and get AI-powered answers.
* 🌍 **Real-Life Example** – Understand concepts through practical, real-world examples.
* 📝 **Quiz** – Test your understanding with AI-generated quiz questions.
* ⚡ **Fast Responses** – Get personalized content in seconds using Google Gemini.

## 🚀 How It Works

1. Enter the topic you want to learn.
2. Select one of the following learning modes:

   * Explain
   * Ask Anything
   * Real-Life Example
   * Quiz
3. Click the generate button.
4. The application sends your request to the Gemini API.
5. The AI generates content based on your selected learning mode.

## ▶️ Running the Project

### Clone the repository

```bash
git clone https://github.com/MeghanaReddyG7/AI_Learning_Buddy.git
```

### Navigate to the project

```bash
cd AI_Learning_Buddy
```

### Install the required packages

```bash
pip install -r requirements.txt
```

### Configure your Gemini API Key

Replace the placeholder API key in the notebook with your own Google Gemini API key before running the application.

### Run the notebook

Open **AI_Learning_Buddy.ipynb** in Google Colab or Jupyter Notebook and execute the cells to launch the Streamlit application.
